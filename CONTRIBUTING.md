# Contributing to repo-atlas

Thanks for your interest! Contributions are welcome.

Please note: we generally don't accept contributions of new skills, and any updates to existing skills must work across all the coding agents we support (Claude Code, GitHub Copilot, Cursor, etc.).

## Process

1. Fork the repository.
2. Switch to the `dev` branch.
3. Create a branch for your work.
4. Submit a PR, being sure to fill in the pull request template.
